## A4 Use

The capsule should answer four questions:

1. What evidence did you use?
2. What boundary did you draw?
3. What did you do to the data or model?
4. What can the final claim not prove?

This is not administrative paperwork. It is the defense layer of the design argument.

## Minimum Capsule

Every A4 package should include:

- one project claim;
- one evidence-input table;
- one system-boundary table;
- one toolchain table;
- one threshold/decision table;
- one roll-forward result label;
- one limits-of-claim statement.

## Jury Standard

A strong A4 does not need maximum computation. It needs a reader to understand how the evidence was made, what changed, what stayed fixed, and why the design action follows from the thermal mechanism.
